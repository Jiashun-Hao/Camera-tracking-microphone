项目简介
此项目为一个根据麦克风预设状态，实现摄像头自动跟踪功能的程序。
系统可根据麦克风的开启状态或预设规则，自动控制摄像头切换与跟踪对应发言人。


Introduction
This project is an automatic camera tracking program based on microphone presets.
The system can automatically control camera switching and track the corresponding speaker according to the microphone status or preset rules.
